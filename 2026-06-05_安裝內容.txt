  打包結果

  ~/kassow_RobotUse.tar.gz（24KB） 

  包含：main.py、src/、requirements.txt、kassow_ros2_api.txt、run.sh、.vscode/
  
  同事安裝步驟：
  1. 解壓：tar -xzvf kassow_RobotUse.tar.gz
  2. 建立 venv：cd kassow_RobotUse && python3 -m venv .venv
  3. 安裝套件：source .venv/bin/activate && pip install -r requirements.txt
  4. 需先安裝 ROS2 Jazzy + 上面兩個 GitHub repo 並 colcon build
  5. 執行：./run.sh
