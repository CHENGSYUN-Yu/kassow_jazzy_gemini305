## 一、後半段手腕相機偵測鏈實作

### 完成項目

**新增物件初始化（auto_grasp.py `__init__`）：**
- `YoloDetectHandcam` — 手腕相機 YOLO（備用，目前流程不使用）
- `GetDepthHandcam` — D405 深度讀取
- `Pixel2HandCam` — 像素→手腕相機座標系
- `HandCam2Flange` — 手腕相機座標系→法蘭座標系
- `Angle2RzHandcam` — 2D 傾角→法蘭座標系 Rz
- `Flange2Base` — 法蘭座標系→base frame（即時 TCP 位姿）
- `WhichFirstHandcam` — 手腕相機優先順序（depth 淺優先，相近時靠左）

**新增流程步驟：**
- `_step_start_handcam_detect()` — 啟動手腕相機偵測階段
- `_process_handcam()` — 座標轉換鏈 + 穩定偵測判斷
- `confirm_handcam` 狀態 — 使用者確認手腕相機結果後進行夾取

**座標轉換鏈（_process_handcam）：**
```
_hand_detector.get_dets()（YoloEngine cam1，D405）
    ↓ GetDepthHandcam.get_depth()
    ↓ Pixel2HandCam.project_all()       → pos_cam_mm
    ↓ HandCam2Flange.transform_all()    → pos_flange_mm
    ↓ Angle2RzHandcam.convert_all()     → yaw_deg, Rz
    ↓ Flange2Base.transform_all()       → pos_base_mm, yaw_base_deg
    ↓ WhichFirstHandcam.get_first()     → 最優先器械
```

## 四、手腕相機改用 YoloEngine（關鍵架構決策）

### 問題
原本 `_process_handcam()` 使用 `YoloDetectHandcam`（subprocess），導致：
1. 模型載入慢（進入 handcam_detecting 時才載）
2. 偵測不到目標（subprocess 推論結果與 GUI 顯示的 YoloEngine 是兩套）

### 解決方案
`_process_handcam()` 改為直接讀取 `_hand_detector.get_dets()`（YoloEngine cam1 已持續在 GPU 推論），不再使用 subprocess。

## 五、D405 深度值異常診斷與修復

### 症狀
```
depth    = 2601 mm   （預期 ~260mm）
pos_base = [504.8, 730.7, -2778.8]   （完全不合理）
y 誤差達 600mm
```

### 診斷過程（嘗試過的方法）

| 方法 | 結果 |
|------|------|
| 加 rs.align(rs.stream.color) 對齊 depth 到 color | ❌ 非主因，但對齊仍有意義保留 |
| 懷疑 D405 固定架設在 2.6m 高處 | ❌ 排除（使用者確認固定在法蘭側邊朝下） |
| 懷疑器械細長導致 bbox 中心穿過 | ❌ 排除（patch_valid=25，25/25 像素都是 2601mm） |
| DEBUG print 輸出 raw depth 和 patch 值 | ✅ 確認 raw=2601mm，patch 全部一致 |
| 查詢 D405 的 depth_units | ✅ **找到根本原因** |

### 根本原因
```
D435I: depth_units = 0.001 m/unit  → 1 unit = 1.0 mm
D405:  depth_units = 0.0001 m/unit → 1 unit = 0.1 mm
```

D405 的 raw 2601 實際上是 **2601 × 0.1 = 260.1mm**，與預期完全吻合。我們把 raw 值直接當 mm 用，放大了 10 倍。

### 修復（realsense.py）
```python
# connect() 成功後從感測器讀取實際 scale
depth_sensor = self._profile.get_device().first_depth_sensor()
self._depth_scale_mm = depth_sensor.get_depth_scale() * 1000.0

# capture_loop 套用
depth_arr = np.asanyarray(depth_frame.get_data()).astype(np.float32) \
            * self._depth_scale_mm
```
