# Kassow KR810 自動夾取系統 — 環境建置說明

## 系統需求

- OS：**Ubuntu 24.04 LTS**
- Python：**3.12**（Ubuntu 24.04 預設）
- ROS2：**Jazzy**
- GPU：NVIDIA（driver 575 以上，支援 CUDA 13）
- 相機：Intel RealSense D435I（頭部）、D405（手腕）

---

## 0. 安裝 NVIDIA Driver

若尚未安裝，執行：

```bash
sudo ubuntu-drivers install
sudo reboot
```

重開機後確認：

```bash
nvidia-smi
# 右上角 CUDA Version 需顯示 13.x
```

---

## 1. 安裝 ROS2 Jazzy

```bash
# 設定 locale
sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8

# 加入 ROS2 apt repo
sudo apt install software-properties-common curl -y
sudo add-apt-repository universe
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key \
    -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

sudo apt update
sudo apt install ros-jazzy-desktop
```

安裝完後確認：

```bash
source /opt/ros/jazzy/setup.bash
ros2 --version
```

---

## 2. 建立 ROS2 Workspace（Kassow SDK）

```bash
# 安裝 colcon（ROS2 不會自動安裝）
sudo apt install python3-colcon-common-extensions

# 在家目錄解壓（tar 內部路徑已包含 ros2_ws/src/）
cd ~/
tar -xzf /path/to/ros2_ws_src.tar.gz

source /opt/ros/jazzy/setup.bash
cd ~/ros2_ws

# 只 build kr_msgs（程式實際只需要這個 package）
colcon build --packages-select kr_msgs --symlink-install
```

建完後將以下兩行加入 `~/.bashrc`（或每次手動 source）：

```bash
source /opt/ros/jazzy/setup.bash
source ~/ros2_ws/install/setup.bash
```

## 3. 安裝 RealSense 相機驅動與中文字型

`pyrealsense2` 的 Python 套件已包含自己的 native library，不需要安裝系統版 librealsense。
只需要安裝 **udev rules**（讓一般使用者有 USB 存取權限）：

```bash
# 加入 Intel RealSense apt repo
sudo mkdir -p /etc/apt/keyrings
curl -sSf https://librealsense.intel.com/Debian/librealsense.pgp | \
    sudo tee /etc/apt/keyrings/librealsense.pgp > /dev/null
echo "deb [signed-by=/etc/apt/keyrings/librealsense.pgp] https://librealsense.intel.com/Debian/apt-repo $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/librealsense.list
sudo apt update
sudo apt install librealsense2-udev-rules

# 加入 plugdev 群組（讓一般使用者可存取 USB 相機）
sudo usermod -aG plugdev $USER

# 安裝中文字型
sudo apt install fonts-noto-cjk
```

安裝後**登出再登入**（或重開機），讓 plugdev 群組與 udev rules 同時生效，再重新插拔相機。

## 4. 設定手臂網路

手臂 IP：`192.168.38.1`

需在手臂所在網路的網卡設定靜態 IP `192.168.38.20`：

```bash
# 查詢連線名稱（NAME 欄位）與裝置名稱（DEVICE 欄位）
nmcli connection show

# 設定靜態 IP（將 <連線名稱> 替換為實際 NAME）
sudo nmcli connection modify "<連線名稱>" \
    ipv4.method manual \
    ipv4.addresses 192.168.38.20/24 \
    ipv4.gateway ""
sudo nmcli connection up "<連線名稱>"

# 確認可 ping 到手臂
ping 192.168.38.1
```

## 5. 解壓主程式並建立 Python venv

```bash
# 安裝 venv 支援（Ubuntu 24.04 不一定預設包含）
sudo apt install python3.12-venv

# 解壓 kassow_handover.tar.gz（建議放在家目錄）
cd ~/
tar -xzf /path/to/kassow_handover.tar.gz
cd kassow_RobotUse

# 建立虛擬環境
python3 -m venv .venv

# 先安裝 PyTorch（需指定 CUDA 13 wheel 來源）
.venv/bin/pip install torch==2.12.0 torchvision==0.27.0 \
    --index-url https://download.pytorch.org/whl/cu130

# 安裝其餘套件（torch 已裝，pip 會跳過）
.venv/bin/pip install -r requirements.txt
```

---

## 6. 啟動系統

```bash
cd ~/kassow_RobotUse
./run.sh
```

`run.sh` 會自動 source ROS2 環境並以 venv 啟動主程式。

## 常見問題

**Q：D405 相機無法開啟**
確認 `librealsense2-udev-rules` 已安裝，重插 USB 或重開機；確認使用者在 `plugdev` 群組（`groups $USER`）。

**Q：手臂無法連線**
確認電腦 IP 已設為 `192.168.38.20`，且 `ping 192.168.38.1` 有回應。

**Q：YOLO 偵測無畫面**
確認相機 index 正確（D435I=0, D405=1），可用 `lsusb | grep Intel` 確認相機是否被系統偵測到。

**Q：torch 安裝後 GPU 不可用**
執行 `.venv/bin/python -c "import torch; print(torch.cuda.is_available())"` 確認；
若為 False，確認 NVIDIA driver 版本 575 以上（`nvidia-smi`）。
